# World-News-Map
A interactive map where you can click on the country for the news!
<img width="1410" height="740" alt="Screenshot 2026-03-01 at 3 22 10 PM" src="https://github.com/user-attachments/assets/ec02c90b-6a49-471e-a39a-6716a77a0657" />
Dependencies you must install
- streamlit
- python (3.14x)
- folium
- pycountry
- reverse_geocoder

Afterwards replace your API key in main.py with your own API key from https://gnews.io/
Once all dependencies are installed run...
streamlit run /Users/your_username/WorldNews-Map/main.py

N.B Some countries do not work because the news api does not have access to those sources
Enjoy!
